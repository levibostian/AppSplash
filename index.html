Documentation

AppSplash is easy. Install it like you would any other WordPress theme. Then all customizing occurs in the Theme Customizer of Wordpress. No plugins or tools necessary.

I should note that this theme uses MailChimp for email address collection. MailChimp is a popular free email marketing solution. The plan is to give the user the ability to change this to app download links or something but that is not the case right now. In order for your website to not be broken, you need to have a MailChimp account.

Installing MailChimp integration.

In order to get MailChimp up and running with this theme, you need to...

  1. Create an account.
  2. Create a MailChimp list used to collect the email addresses with.
  3. Open up the list and go to the "Signup forms" page.
  4. Go to the "Embedded forms" page.
  5. In the bottom right part of the page, you should see a box that says "Copy/paste onto your site". Inside of this box you will find text that reads: 
                                        
        <form action="//example.us10.list-manage.com/subscribe/post?23942=3948398&id2093">
    or something like that. Copy just the part inside of the double quotes and paste that into the Theme Customizer option "MailChimp information".
